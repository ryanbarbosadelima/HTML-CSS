Negrito e Itálico

Negrito:

Nessa frase temos um termo em negrito utilizando a tag "b" que é uma tag não semântica

Aqui, já termos um termo em negrito com a tag "strong" que é semântico, pois além de dar o formato, é responsável também por dar um significado

Itálico:

Nesses casos se utilizava a tag "i" e esse era o resultado, entretanto ela se tornou obsoleta por não ser semântica

Atualmente é mais comum que se utilize a tag "em" para que se coloque os textos em itálico

É importante que se entenda que as tags não semânticas só são obsoletas pois elas dão forma mas não sentido

Podemos também utilizar a tag "mark" para adicionar um texto marcado

Textos Grandes e Pequenos

Texto Grande se utilizava a tag "big" porém caiu em desuso, porém texto pequeno ainda se utiliza a tag "small", entretanto é mais comum que se utilize da CSS para que determine o tamanho das letras

Texto deletado

podemos marcar como deletado ou excluído um texto que não deve ser lido com a tag "del"

Texto inserido

podemos marcar como inserido um texto que foi adicionado na página posteriormente, com a tag "ins" ficando sublinhado

Texto Sobrescrito

Para subscrever textos é utilizado a tag "sup" para que seja colocado em cima e "sub" para ser utilizado embaixo, é normalmente utilizado em exemplos como: X20+3 ou até mesmo H20
