Hierarquia da Tag H

essa é a a segunda linha de hierarquia, sendo o "h2"

aqui seria a terceira sendo o "h3"

a 4ª aqui com o "h4"

5ª com o "h5"

Não existe nem h0 nem h7, essas são as hierarquias dentro do código html, para melhor compreensão é como se fosse um sumário de um livro, contendo elementos que devem ser respeitados para que haja uma coerência
