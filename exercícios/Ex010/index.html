Usando Links Externos

Exemplo link que fecha essa aba e me joga pra outro site: Veja receitas no link a seguir receitas online

Exemplo link outra aba: Veja receitas no link a seguir: site para receitas online

Por que isso ocorre?

Isso acontece pois dentro da tag "a" é necessário que se coloque alguns elementos como o "target="_blank"" e o "rel="external""

Links Internos

É possível também que se acesse páginas dentro do seu próprio site, inicialmente é necessário que se abra um novo arquivo dentro da mesma pasta deste. Por exemplo temos agora a nossa SEGUNDA PÁGINA

TERCEIRA PAGINA

É importante destacar que quando as páginas seguintes forem do próprio site existem alguns complementos dentro da tag "a", como por exemplo a utilização do "target="_self""", o "rel="next"" para páginas posteriores e o "rel="prev"" para páginas anteriores

Links para Download

Geralmente esses links para download utilizam formatos de arquivos diferentes, é importate então estar bem atento ao formato do arquivo que irá disponibilizar para baixar quando for colocá-lo no código.
