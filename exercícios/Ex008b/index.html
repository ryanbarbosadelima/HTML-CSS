Código fonte e Pré-Formatação

O comando document.getElementByld('teste') é escrito em linguagem JavaScript.

A mudança de "fonte" na frase acima é utilizado a tag "code" que é ideal para demonstrar linhas de código pois os caracteres se transformam em uniespassados, ou seja, ocumpam o mesmo espaço ocasionando uma melhor visualização.Entretanto somente a tag "code" não permite que a linha de código fique organizada. Exemplo:

num = int(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é PAR) else: print(f'0 número {num} é ÍMPAR) print('Fim do programa')

Para que se organize com as quebras linhas da maneira certa é usar a tag "pre"

        
        num = int(input('Digite um número'))
        if num % 2 == 0:
            print(f'0 número {num} é PAR)
        else:
            print(f'0 número {num} é ÍMPAR)
        print('Fim do programa')
